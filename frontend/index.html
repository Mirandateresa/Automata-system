Acerca de los Autómatas

Este sistema implementa diferentes autómatas utilizando:

  • Estructura CASE/SWITCH: Para definir transiciones entre estados
  • Estructura WHILE: Para recorrer las cadenas de entrada
  • Estados finitos: Cada autómata tiene estados definidos

Código de Ejemplo

// Estructura CASE en el autómata
switch(estado) {
  case 0:
    if (simbolo === '1') estado = 1;
    break;
  case 1:
    if (simbolo === '1') estado = 0;
    break;
}

// Estructura WHILE para recorrer
let i = 0;
while(i < cadena.length) {
  // Procesar símbolo
  i++;
}
